# Movie Search & Rating App

Movie search and rating app using TMDB.


## Requirements

- Node.js v16+
- Python **3.11.9**
- npm


## Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```


## Frontend

From project root:

```bash
npm install
npm run dev
```


## Usage

* Search movies by title
* Click a movie to see details
* Rate movies from 1 to 5 stars


## Tech Stack

* **Frontend:** React, TypeScript, Tailwind, Vite
* **Backend:** Python 3.11.9, Flask
* **Database:** Supabase
* **API:** TMDB API
